# ESPHome e-ink Dashboard for Home Assistant
My battery powered e-ink dashboard for Home Assitant using ESPHome!

Hardware supported out of the box by this project: 
* LilyGO T5 4.7" (gen 1)
* Seeed Studio reTerminal e1001

More detailed information about this project can be found from creatingsmarthome.com blog
* LilyGO t5 4.7" guide: https://www.creatingsmarthome.com/?p=1043
* Seeed Studio reTerminal e1001: https://www.creatingsmarthome.com/?p=4743
